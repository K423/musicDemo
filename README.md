# musicDemo
easy musicDemo

一个简单的音乐网站后台管理系统 内容包括系统首页、用户管理、歌手管理与歌曲管理


开发技术设计springboot、vue、mybatis、mysql和maven项目管理工具等

(个人开发，页面美观方面一般般，主要关注点在逻辑实现方面)
